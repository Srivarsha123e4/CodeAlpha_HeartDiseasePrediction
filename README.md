# Heart Disease Prediction using Machine Learning

## Project Overview

This project predicts the possibility of heart disease using Machine Learning algorithms based on patient medical data.

## Technologies Used

* Python
* Pandas
* NumPy
* Scikit-learn

## Machine Learning Algorithm

* Random Forest Classifier

## Dataset

Heart Disease dataset from UCI Machine Learning Repository.

## Features Used

* Age
* Sex
* Chest Pain Type
* Blood Pressure
* Cholesterol
* ECG Results
* Heart Rate
* Exercise Induced Angina
* And more medical attributes

## Project Workflow

1. Data Collection
2. Data Preprocessing
3. Handling Missing Values
4. Feature Scaling
5. Model Training
6. Disease Prediction
7. Accuracy Evaluation

## Output

The system predicts whether a patient has a high or low chance of heart disease.

## Accuracy

Achieved approximately 98% accuracy using Random Forest algorithm.
